# ✈️ SkyRoute API

API REST desarrollada con **Django REST Framework** para la gestión de **Vuelos** y **Aerolíneas**.

## ⚙️ Tecnologías Usadas
- Python 3
- Django 5
- Django REST Framework

## 🚀 Instrucciones de Ejecución
```bash
git clone 
cd skyroute_api
python manage.py runserver
